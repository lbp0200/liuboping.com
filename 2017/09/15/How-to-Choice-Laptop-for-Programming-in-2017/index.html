How to Choice Laptop for Programming in 2017

Memory

as a programmer, you have lots of software tools to help you, such as IDE(eclipse, WebStorm, PHPStrom…), the database for development (MySQL, PostgreSQL, Redis) and web browser (chrome). all these things use tons of memory. so the RAM size must be enough to run all these things. in my expression, 16G or bigger is recommended.

CPU

intel I3, I5, I7 usually is enough, based on your money.

Hard Disk

SSD is recommended. if you use Hadoop, at least 1T HDD is required.

Graphics Card

Integrated Graphics is recommended. if you develop machine learning you should choose NVIDIA.

Weight

the lighter the better

Screen

IPS and 4K is recommended.
